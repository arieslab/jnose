Downloading and running JNose Test
===================================

Step by step
-------------------------

``git clone https://github.com/tassiovirginio/jnose.git``
 
 ``cd jnose ``

JNose Test has a local user interface

``mvn jetty:run``

Then, access the localhost on your browser ``http://127.0.0.1:8080``.
